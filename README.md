# Prueba Técnica – Automatización e-Ship

**Candidato:** Camilo García Rojas  
**Stack:** Python 3.10 + SQLite  

## Requisitos
- Python 3.10 o superior

## Ejecución
1. Clonar el repositorio.
2. Ubicarse en la carpeta raíz.
3. Ejecutar: `python src/main.py`

## Estructura
- `src/` – Código fuente
- `data/` – Archivos JSON (solicitudes, pedidos)
- `logs/` – Logs de procesamiento

## Supuestos
- Formato de ID: `OPaammdd-12345678` (fecha válida)
- Ventana de deduplicación: 2 horas
- Aprobación manual simulada por consola
